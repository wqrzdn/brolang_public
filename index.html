Language Basics

BroLang is a slang-inspired programming language that makes coding more accessible and fun.

Variables

Output

Loops

wloop (condition) { /* code */ }

While loop - repeats as long as condition is true

yappin (1 to 5) { /* code using i */ }

Conditionals

Return Values

About BroLang

The Story

BroLang was created to make programming fun, accessible, and relatable through familiar slang expressions. It reduces the entry barrier for new programmers while providing a memorable and engaging coding experience.

Technical Specs

BroLang is implemented as a JavaScript interpreter with a tokenizer and parser. It runs directly in the browser, making it perfect for educational purposes and quick prototyping.

Technology Stack:

  • Frontend: HTML, CSS, JavaScript
  • Interpreter: Custom tokenizer and recursive descent parser
  • Execution: Client-side JavaScript runtime
